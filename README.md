# ⚔️ Guild Clash Optimizer

Калькулятор оптимальной расстановки для гильдейских схваток (Guild Clash) в Hero Wars: Alliance.

## Что делает

Распределяет игроков гильдии по 3 линиям (TL / ML / BL) так, чтобы захватить максимум флагов (2 из 3 = победа).

## Как работает

### Механика горки
- Игроки на каждой линии сортируются слабый → сильный
- Каждый бьётся до 2 раз: победитель остаётся, проигравший заменяется
- Последний бой определяет флаг
- Равная мощность = проигрыш атакующего (преимущество защиты)

### Алгоритм оптимизации (5 фаз)
1. **Эвристики** — 4 стратегии выбора жертвы × 6 стратегий распределения × все размеры линий
2. **Свапы 1-на-1** — обмен игроков между линиями (до 10 раундов)
3. **Свапы 2-на-2** — обмен пар (до 3 раундов)
4. **Перемещения** — перенос игрока между линиями, меняет размер (до 10 раундов)
5. **Повторные свапы** — после перемещений (до 5 раундов)

**Приоритет:** флаги > побед > меньше рисков

### Страховочный расчёт (Insurance)
Тот же пул и алгоритм, но оценка приоритизирует **robust flags** — флаги, выдерживающие 1 случайный апсет (убираем самый тонкий выигрыш, пересимулируем).

Приоритет: robust flags → flags → wins → risk

### Анализ
- **Робастность** — 6 перестановок вражеских блоков (ТОП-10 / СЕР-10 / СЛБ-10)
- **Точечные свапы** — переброска каждого из ТОП-5 врага на другую линию
- **Вероятностная модель** — бои с маржой < 5% считаются рискованными (50-100% шанс)
- **Слепой режим** — 4 модели распределения × 3 линии = 64 сценария

## Интерфейс

### Наша команда
- 3 таблицы × 10 строк
- Поля: имя, атака, защита, тотал (авто-сумма)
- Сортировка по мощности — кнопка ↕ Sort
- Enter: Name → Name (между именами), Atk → Def → Atk (между мощностями)
- Игроки с total=0 не участвуют в расчёте (затемняются после Calculate)

### Враги (5 раундов)
- Два режима: Разведка (знаем каждого) / Слепой (только тотал линии)
- Поля на линию: Attack, Defense, Total (информационные) + список мощностей
- Авто-сортировка по возрастанию
- Drag-and-drop между линиями
- Enter: TL.Atk → TL.Def → TL_1 → ... → TL_10 → ML.Atk → ...
- Первый раунд — слепой по умолчанию

### Результаты
- Normal / 🛡 Insurance / ⚖ Compare — переключатель
- Вкладки: Fights, Robustness, Fine swaps, Lineups (разведка) / Fights, Scenarios, Lineups (слепой)
- Метрики: Split, Flags, Wins, Reliability, 🛡 Robust
- Секции Our Team и Enemy сворачиваются (авто после расчёта)

### Данные
- **localStorage** — автосохранение при каждом изменении
- **Export** — JSON-файл со всеми данными (наши + 5 раундов врагов)
- **Import** — загрузка из JSON-файла

## Использование

Открыть `index.html` в браузере. Один файл, нет зависимостей, работает офлайн.

1. Заполнить нашу команду (имена + мощности)
2. Переключиться на раунд, ввести врагов
3. Нажать Calculate
4. Смотреть результат, переключать Normal / Insurance / Compare

## Структура файла

Один файл `index.html` — HTML + CSS + JavaScript. Без фреймворков, без сборки.

## Константы

| Константа | Значение | Описание |
|-----------|----------|----------|
| `RISK` | 0.05 | Порог зоны риска (5% маржа) |
| `INS_UPSETS` | 1 | Количество апсетов для страховки |
| `NR` | 5 | Количество раундов |
| `LN` | TL, ML, BL | Названия линий |

## Единицы измерения

Значения вводятся и хранятся как есть. Суффикс **K** в отображении результатов — чисто визуальный маркер, конвертаций нет.
