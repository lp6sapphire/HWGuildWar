⚔️ GW Planner
📖 Инструкция

1. Ростер — введите имена и силу героев (⚔️) и титанов (⚡) каждого игрока. E5 = полный элемент титанов. 🚧 = рискованный игрок (не ставить на ключевые здания). Кнопки ± для быстрой корректировки.

2. Enemy (враг) — введите силу защитников каждого здания. Для титанов отмечайте E5 чекбоксом. Не-E5 враги получают штраф (Non-E5 penalty в настройках).

3. Настройки ⚙️

• Pts per win — очки за победу (обычно 20)
• E5 boost — бонус силы E5 титанов (%)
• Non-E5 penalty — штраф не-E5 титанов (и наших, и врагов)
• Hero margin — мин. запас силы для героев (%)
• Titan margin — мин. запас силы для титанов (%)

4. Ghost Bridge 👻 — если включен, алгоритм обязательно берёт Казармы + Маяк + Академию Магов (страховка пути к T2).

5. ⚔️ Calc — запускает расчёт. Сбрасывает прогресс (spent/closed). Генерирует два плана:

• 🗡 Max — агрессивный: использует все возможные победы, включая risky (маржа ниже порога, но побеждаем)
• 🛡 Safe — страховочный: только уверенные победы (маржа выше порога). Risky атаки отбрасываются. Может потерять здания, но каждая атака надёжна.

6. Трекинг боя — в таблице игроков кликайте 0/1/2 (потраченные атаки). В таблице зданий кликайте ✓ (слот закрыт). План автоматически пересчитывается.

7. Алгоритм

• Перебирает 1024 комбинации зданий, выбирает максимум очков
• R1: герои на hero-здания (сильнейший на дефицитный слот)
• R2: титаны на titan-здания (слабейший достаточный)
• R3-R4: оставшиеся атаки
• Swap: выравнивает маржу, группирует 2 атаки на одно здание

8. Экспорт/Импорт — 💾 сохраняет JSON, 📂 загружает. Данные также сохраняются в браузере автоматически.

1. Roster — enter player names and hero (⚔️) / titan (⚡) power. E5 = full element titans. 🚧 = risky player (avoid on key buildings). Use ± buttons for quick adjustments.

2. Enemy — enter defender power for each building slot. Mark E5 checkbox for titan buildings. Non-E5 enemies receive a penalty (Non-E5 penalty setting).

3. Settings ⚙️

• Pts per win — points per fight won (usually 20)
• E5 boost — E5 titan power bonus (%)
• Non-E5 penalty — penalty for non-E5 titans (ours and enemy)
• Hero margin — min power margin for heroes (%)
• Titan margin — min power margin for titans (%)

4. Ghost Bridge 👻 — when enabled, algorithm must capture Barracks + Lighthouse + Mage Academy (T2 path insurance).

5. ⚔️ Calc — runs calculation. Resets progress (spent/closed). Generates two plans:

• 🗡 Max — aggressive: uses all possible wins, including risky ones (margin below threshold but still winning)
• 🛡 Safe — conservative: only confident wins (margin above threshold). Risky attacks are skipped. May lose buildings, but every attack is reliable.

6. Battle tracking — click 0/1/2 in player table (spent attacks). Click ✓ in building table (slot closed). Plan auto-recalculates.

7. Algorithm

• Enumerates 1024 building combinations, picks max score
• R1: heroes on hero-buildings (strongest on scarcest slot)
• R2: titans on titan-buildings (weakest sufficient)
• R3-R4: remaining attacks
• Swap: evens out margins, consolidates 2 attacks on same building

8. Export/Import — 💾 saves JSON, 📂 loads. Data also auto-saves in browser.
